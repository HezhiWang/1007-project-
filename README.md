## Dependencies
- Python = 3.5
- tkinter == 8.5.18
- geopy == 1.11.0
- gmplot == 1.1.1
- seaborn == 0.7.1
- numpy == 1.11.1
- pandas == 0.18.1
- matplotlib == 1.5.3
- sklearn == 0.18.1

if any of the packages are missing, type in your command line(MAC OS X)
```sh
$ pip install <package>==<version>
```

## How to run

Switch to the directory of the application, and run on command line(MAC OS X):
```sh
$ python gui.py
```
